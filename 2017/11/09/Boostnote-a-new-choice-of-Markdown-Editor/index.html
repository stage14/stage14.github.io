Markdown 编辑器的另类选择——Boostnote

日期： 2017-11-09 | 分类： Applications | 作者：Chris HE

一如我们「听过很多人生道理，依然过不好一生」，我们看过很多评测，依然选不出一个自己喜欢的 Markdown 编辑器。

相对于其他工具，Markdown 编辑器对我来说并不是常用的，目前用得比较多的也就是 Bear 和 MWeb Lite。Bear 的颜值已经得到用户的认可，就算是是免费使用功能也是足够我使用；国产的 MWeb 绝对是一个功能极其强大的存在，甚至是很多人写 blog 的御用 Markdown 编辑器，而 Lite 免费版本也是完全够我平常使用。偶遇 Boostnote 并开始试用，一方面有好奇心驱使，一方面是简单、利落的界面。

Boostnote is an Open source note-taking app for programmers.

官方将 Boostnote 定位在一个给程序员使用的工具，除去最重要的 Markdown 编辑功能，代码片段管理似乎也是一把好手，不过能否怼 Quiver 就留给程序员去判断了。

试用下来之后有两个地方让我敲字的时候有愉悦感，一个是半全屏模式，右上角一个小按钮开关就可以让几乎整个界面变成一张「稿纸」，可以专注的敲字上；另外一个就是「Quick Preview」，全心码字的时候可以锁掉这个功能，基本成文之后可以解锁这个快速预览功能，这样对文章修改和调整的效率有着近乎质的提高。这篇文字还没码完，我就有点回不去 Atom 那「Control+Shift+M」的「快速预览」了 😂😂

像标签管理、个性化界面等功能虽然比较方便，但是并不十分完善；另外还有不少「程序员友好」功能，尝试下来也并非十分强健，是不是真的够程序员小伙伴用就不得而知。

Boostnote 还支持 Dropbox、GoogleDrive 等云服务，可以实现桌面版本和移动端版本的同步——是的，还有 iOS 和 Android 版本；Android 版本我不知道，但是 iOS 版本目前真的很弱鸡，iPad 版本上面新建文件的按钮都没有局中对齐……
